# Mech-488-maze

A C++ project for mechanical engineering course Mech-488.

## Components

1. maze
2. joystick
3. checkpoint
4. button

## Building

This project uses a Makefile for compilation. Make sure you have `clang++` installed.

```bash
make        # Build the project
make run    # Build and run
make clean  # Remove build artifacts
```

## Project Structure

```
src/       - Source files (.cpp)
include/   - Header files (.h)
bin/       - Build output directory
```
